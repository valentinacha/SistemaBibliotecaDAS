# Sistema de Gestion de Biblioteca
Aplicacion de escritorio desarrollada en C# para administrar libros, usuarios y prestamos mediante operaciones CRUD.

## Tecnologias Utilizadas 
- C#
- Windows Forms 
- .NET
- Visual Studio 

## Instalacion 
1. Clonar el repositorio 
2. Abrir el proyecto en Visual Studio 
3. Ejecutar el proyecto 

## Uso 
El sistema permite: 
- Administrar Libros 
- Administrar usuarios 
- Registrar prestamos 
- Visualizar estadisticas
